The Portfolio Dummy Website

This is a sample page for rapid prototying and simple routing defined by a xml. You can navigate through XML file items and update the page via ajax and json data.

Feature List

  • supporting back button via history.hs
  • dynamically adding new content via ajax/json requests
  • very simple description and adding of new sites through
    nested xml elements
  • inheritance of defined attributes to child elements
  • one page template to rule them all
  • root means home
  • CSS3 and HTML5 layout
  • responisiveness and nice degration (in the future)
